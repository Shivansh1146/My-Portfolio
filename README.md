# 🚀 Shivansh Jaiswal - Personal Portfolio

A sleek, premium, and lightning-fast personal portfolio built using **React** and **Vite**. 

## ✨ Features
- **Tokyonight Glassmorphism:** A beautiful dark-mode aesthetic with frosted glass panels.
- **Dynamic Projects Showcase:** Easily highlight top projects like MedicSense AI and Study Buddy.
- **Responsive Layout:** fully responsive across all mobile and desktop devices.
- **Resume Integration:** Integrated PDF hosting for one-click resume downloads.

## 🛠️ Tech Stack
- React
- Vite
- Custom CSS (Glassmorphism UI)

## 🏃‍♂️ Run Locally
1. Clone the repository: `git clone https://github.com/Shivansh1146/Shivansh-Portfolio.git`
2. Run `npm install`
3. Run `npm run dev` to start the local server.
